# Юмор во времени

«Юмор во времени» - практический сайт о бережном использовании юмора для самопомощи, работы с эмоциями и психологической поддержки.

Сайт больше не является статичной презентацией. Он помогает пользователю выбрать состояние, подобрать практику, пройти небольшой тест на стиль юмора и посмотреть инструменты для психологов.

## Назначение сайта

- дать короткие упражнения с юмором для снижения напряжения;
- помочь подобрать практику по состоянию, цели, аудитории и времени;
- показать безопасные и небезопасные способы использования юмора;
- дать психологам карточки, вопросы и формулировки для работы с клиентами;
- сохранить историю юмора как дополнительный контекст, а не центральную часть проекта.

Практики на сайте не заменяют психолога, психотерапевта, врача или экстренную помощь.

## Технологии

- React 18;
- TypeScript;
- Vite;
- lucide-react;
- Vitest;
- ESLint;
- pnpm.

## Требования

- Node.js 20 или новее;
- pnpm 9 или новее.

## Установка

```bash
pnpm install
```

## Локальный запуск

```bash
pnpm dev
```

По умолчанию Vite запускается на `http://127.0.0.1:5173`. Если порт занят, Vite предложит другой порт.

## Проверки

```bash
pnpm lint
pnpm typecheck
pnpm test
```

## Production-сборка

```bash
pnpm build
```

Собранные файлы появляются в `dist/`.

## Предпросмотр production-сборки

```bash
pnpm preview
```

## Структура проекта

- `src/App.tsx` - основной интерфейс сервиса;
- `src/data/service.ts` - практики, тест, инструменты психолога, история и источники;
- `src/lib/serviceLogic.ts` - алгоритмы подбора практик и подсчета результата теста;
- `src/lib/serviceLogic.test.ts` - тесты ключевой логики;
- `src/components/Header.tsx` - навигация сайта;
- `src/styles.css` - визуальная система и адаптивность;
- `public/404.html` - страница ошибки 404.
